# Otter Clone

React clone of the Otter landing page (withotter.com).

## Stack

- React 18
- Vite 6
- Tailwind CSS v4

> Tailwind v4 doesn't work with Vite 7+, keep vite at ^6.3.5

## Setup

```bash
npm install
npm run dev
```

Opens at http://localhost:5173

## Scripts

```bash
npm run dev       # dev server
npm run build     # production build → dist/
npm run preview   # preview the build
```
